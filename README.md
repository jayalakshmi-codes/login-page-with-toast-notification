# Login Page with Toast Notification

## Description
This project is a simple and responsive login page designed with HTML, CSS, and JavaScript. The login form is centered on the screen, providing a clean and user-friendly interface.

The application also includes toast notifications to display success or error messages based on the login validation.

## Features
- Centered login form layout
- Simple and clean user interface
- Success and error toast notifications
- Basic form validation using JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How It Works
1. Users enter their login credentials.
2. JavaScript validates the input fields.
3. A toast notification is displayed to show whether the login is successful or if there is an error.

## Purpose
This mini project was developed to practice form design, UI alignment, and JavaScript-based notifications.
